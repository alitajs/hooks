# @alitajs/hooks

react hooks library

[![NPM version][image-1]][1] [![NPM downloads][image-2]][2]

## Install

```
// npm
npm install @alitajs/hooks --save

// yarn
yarn add @alitajs/hooks
```

## Usage

```
import { useToggle } from '@alitajs/hooks';
```

## Development

```
$ git clone git@github.com:alitajs/hooks.git
$ cd hooks
$ npm install
$ npm start
```
Open your browser and visit http://127.0.0.1:8001/hooks , see more at Development.
